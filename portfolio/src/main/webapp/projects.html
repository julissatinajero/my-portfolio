Imminent Reprisal

 

(Game)

Imminent Reprisal is a choose-your-own adventure game inspired by Greek mythology! Users are presented with a series of choices to dictate the protogonist's fate. The game requires users to sign in using a Gmail account in order to keep track of the attempts, or "lives", the user has left to complete the storyline.

(Educational)

FUNdamentals is an educational platform for young students who want to learn some of our everyday basics! This colorful and fun webapp has three learning models where students can practice the alphabet, colors, and numbers. There is a small game incorporated to each module to create a more interactive environment as well!
